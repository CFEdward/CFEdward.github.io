Features

One of the very first projects I created in order to learn Unreal Engine, made entirely with Blueprint scripting.


It's a very simple game, you control a space marine from a top-down perspective defending against unrelenting waves of enemies. This project helped me to get familiar with the editor and core concepts like gameplay mechanics, basic game flow and some basic UI.
